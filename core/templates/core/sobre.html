Sobre o Projeto Immerse

O Problema

No cenário globalizado, a proficiência em inglês tornou-se uma competência fundamental. Apesar disso, o Brasil apresenta índices modestos de fluência, apontando para barreiras significativas no aprendizado. O problema central que motiva este projeto é a dificuldade encontrada por estudantes brasileiros para adquirir fluência em inglês, com um notável descontentamento com métodos de ensino tradicionais, que muitas vezes falham em preparar os alunos para o uso prático da língua.

Nossa Solução

Este trabalho propõe o desenvolvimento da "Immerse", uma plataforma web educacional que facilita o aprendizado de inglês por meio de conteúdos audiovisuais (vídeos) e questionários interativos. A plataforma é fundamentada na teoria de aquisição de segunda língua de Stephen Krashen, que diferencia o "aprender" consciente de regras do "adquirir" subconsciente da língua, similar ao que ocorre com falantes nativos.

Metodologia e Tecnologia

O desenvolvimento seguiu uma metodologia de três etapas: "Ouvir e Interpretar" o problema, "Criar e Prototipar" a solução e "Implementar e Testar" para validação. A plataforma foi construída utilizando Python com o framework Django para o back-end, MySQL como banco de dados e JavaScript para garantir a interatividade no front-end.

Este Relatório Técnico-Científico foi apresentado na disciplina de Projeto Integrador para o curso de Bacharelado em Ciência de Dados da Universidade Virtual do Estado de São Paulo (UNIVESP).

Autores: Adriano Dos Santos Paiva e Emerson Nunes Junqueira.
